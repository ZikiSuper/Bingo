SUGESTÕES DE MELHORIAS SEGURAS PARA O BINGO

=== 1. PERFORMANCE (Sem Impacto no Sorteio) ===

✅ Usar Set em vez de Array para `sorteados`
   - Problema atual: generateRandomNumber() faz O(n) checks
   - Solução: usar Set para O(1)
   - Impacto: ZERO no funcionamento visual/de sorteio
   
   CODE:
   // ANTES
   const state = {
     sorteados: []
   };
   while (state.sorteados.includes(number)); // O(n)
   
   // DEPOIS
   const state = {
     sorteados: new Set()
   };
   while (state.sorteados.has(number)); // O(1)
